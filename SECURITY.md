# Security Policy

## Supported Versions

This is a static portfolio website. Security fixes and updates are applied to the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please report it privately.

**Preferred contact**

- Email: contact@zyinnova.com
- Subject: `Security report — portfolio.zyinnova.com`

## What to include in your report

Please include:

- A clear description of the issue
- Steps to reproduce (proof of concept if possible)
- Impact assessment (what could happen)
- Screenshots/logs (if helpful)
- Your contact info for follow-up

## Response timeline

I will acknowledge your report as soon as possible and work on a fix. If the issue is confirmed, I will prioritize a patch and deploy the update.

## Scope

In scope:

- `portfolio.zyinnova.com` (static site)
- Client-side code in this repository

Out of scope:

- Issues caused by third-party services outside this repo
- Social engineering
- Denial-of-service attacks
