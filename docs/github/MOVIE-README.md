# Movie Explorer

A collaborative Flutter learning prototype that retrieves movie information
from The Movie Database (TMDB) API.

## Verified scope

- Flutter and Dart application
- HTTP integration with the TMDB API
- Environment-based API configuration with `flutter_dotenv`
- Localization packages and language resources in the repository

The repository does not currently provide evidence for production deployment,
user accounts, Firebase, favorites, comments, ratings, notifications or store
publication. Those features are therefore not claimed here.

## Collaboration

This was developed as a collaborative prototype by Ziyaaddin and Muhammed. The
repository is retained as a learning and integration example rather than
selected founder-product work.

## Run locally

1. Install a compatible Flutter SDK.
2. Obtain a TMDB API key.
3. Configure the environment file expected by the application without
   committing the key.
4. Run:

   ```bash
   flutter pub get
   flutter run
   ```

TMDB attribution: This product uses the TMDB API but is not endorsed or
certified by TMDB.
