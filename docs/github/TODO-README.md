# Multi-language Todo App

A public Flutter/Firebase prototype for managing personal tasks in English and
Finnish.

## Verified scope

- Email/password authentication and password reset with Firebase Authentication
- User-specific tasks stored in Cloud Firestore
- Task priorities, due dates and filtering
- Local notifications
- English and Finnish localization
- Device-local preferences

## Status

This repository is a public prototype and source-code example. It is not
presented as a production service or a currently hosted live application.

## Technology

- Flutter and Dart
- Firebase Core, Authentication and Cloud Firestore
- Flutter local notifications
- Flutter localization
- Shared preferences

## Run locally

1. Install a compatible Flutter SDK.
2. Configure a Firebase project for the target platform.
3. Add the platform Firebase configuration files locally.
4. Run:

   ```bash
   flutter pub get
   flutter run
   ```

Do not commit private keys, service-account credentials or personal
configuration files.
